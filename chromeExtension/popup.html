Privacy Policy

KungFlow collects work-pattern metrics only to help estimate cognitive load and improve focus-related features inside the application.

The extension may send metrics such as tab switches, open-tab count, keyboard activity counts, typing speed, mouse movement speed, login/session details, and cognitive-load status to the KungFlow server.

KungFlow is committed to using this data only for the purpose of running the service, showing user status, improving focus recommendations, and maintaining account security. KungFlow promises not to misuse the data sent to it, sell it, or use it for unrelated tracking.

KungFlow does not intentionally collect page content, passwords, private messages, or the text typed by the user. The metrics are numeric activity signals, not the actual content of the user's work.

Users should run KungFlow only with a trusted server endpoint. If the local server is stopped or unreachable, metrics may remain local until the application can sync again.
